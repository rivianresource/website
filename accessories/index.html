Charging Accessory

A2Z CCS1 to NACS Adapter

First impressions:

The adapter arrived nicely packaged and includes a solid storage case. The adapter itself feels heavy, durable, and well made — definitely not cheap feeling.

Real-world usage:

For Rivian owners with a NACS port, a CCS1 to NACS adapter is pretty much essential right now. I’ve already used this several times at my local Rivian Adventure Network stations since all of them are currently set up as CCS1.

I keep it stored in the frunk so it’s always easy to grab when needed. Usage is straightforward. From what I understand, the adapter should be connected to the charging cable first before plugging it into the vehicle.

Every time I’ve used it so far, the Rivian recognized the charger immediately and charging started right away without any issues.

I decided to go with the A2Z version because of the strong feedback from other Rivian owners, and it’s also a bit less expensive than the Rivian-branded adapter, which currently retails around $200.

One thing worth mentioning: if an adapter causes damage to the vehicle, Rivian states that damage would not be covered under warranty. Personally, I view that as a pretty unlikely scenario, but it’s still important information to share.

Verdict:

Highly recommend.
